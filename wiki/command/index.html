towny

1. /t,/n

・街作成(2000)：/t new 町名
・国作成(15000)：/n new 国名
・街へ参加(招待されていない場合はできません)：/t join 町名
・街の情報を確認(街に参加している場合)：/t
・国の情報を確認(国に参加している場合)：/n
・街のスポーン地点へテレポート：/t spwan
・国のスポーン地点へテレポート：/n spawn
・町長の変更：/t set mayor 名前
・国王の変更：/n set king 名前
・町名の変更：/n set name 町名>br> ・国名の変更：/n set name 国名
・/t系ヘルプ：/t ? ・/n系ヘルプ：/n ? ・街の銀行に入金する：/t deposit 値段
・国の銀行に入金する：/n deposit 値段
・街の銀行から出金する：/t withdraw 値段
・国の銀行から出金する：/n withdraw 値段
・今いるチャンク(土地,16✕16)を500で街の領土にする：/t claim
・常時、いるチャンクを街の領土にする：/res toggle townclaim on/off
・今いるチャンク(土地,16✕16)を街の領土を手放す：/t unclaim
・取得できるチャンク上限を上げる(街の人数によって上限が変わるので無効です)：/t buy bonus 値段
・街を削除する：/t delete 町名
・国を削除する：/n delete 国名
・街の領土内での権限を変更する(build:設置、destroy:破壊、switch:ボタン,スイッチの使用、itemuse:エンダーパールなどのアイテム使用：
/t set perm build on/off
/t set perm destroy on/off
/t set perm switch on/off
/t set perm itemuse on/off
・街の権限を変更する：/t rank add/remove 名前 ランク名
・国の権限を変更する：/n rank add/remove 名前 ランク名
・街の住民の権限を表示する：/t ranklist
・国の住民の権限を表示する：/n ranklist
・trust権限(土地の権限関係なく設置等ができる権限)を与える(住民でなくても可)：/t trust 名前
・街の領土内での設定を変更する(explosion:爆発による破壊、fire:火の延焼,mobs:敵対mobのスポーン,pvp:プレーヤーへの攻撃)
/t toggle explosion on/off(on/off関係なく爆破による破壊はロールバックされます)
/t toggle fire
/t toggle mobs
/t toggle pvp
・国への所属を街へ招待/除外する：/n add/kick 町名
・同盟の申請/解除：/n ally add/remove 国名
・指定した国のスポーン地点にテレポートする：/n spawn 国名
・敵対関係の登録/解除：/n enemy add/remove 国名
・中立国になる：/n toggle neutral on/off
・国の首都を変更する：/n set capital 町名
・指定した国民に称号を付ける：/n set surname 名前 称号

2. /plot

・今いるチャンクを売る(街の領土内)：/plot fs 値段
・今いるチャンクの販売を中止する：/plot nfs
・売られているチャンクを買う：/plot claim
・チャンクの情報を確認する：/plot perm
・チャンクの権限を変更する(resident:街の住民、ally:同盟国の住人、outsider:その他全員)
(build:設置、destroy:破壊、switch:ボタン,スイッチの使用、itemuse:エンダーパールなどのアイテム使用)：
/plot set perm 立場 権限 on/off
例(outsiderから全ての権限を剥奪)：/plot set perm outsider off
・チャンクをアリーナにする(プレーヤーへの攻撃ができる)：/plot set arena
・チャンクの設定を変更する(explosion:爆発による破壊、fire:火の延焼,mobs:敵対mobのスポーン,pvp:プレーヤーへの攻撃)
/plot toggle explosion on/off(on/off関係なく爆破による破壊はロールバックされます)
/plot toggle fire
/plot toggle mobs
/plot toggle pvp

3. /res

・プレーヤーの情報を見る：/res 名前
・フレンドリストに追加する(相手の友達リストには入らない)：/res friend add/remove 名前
・フレンドリストを見る：/resident friend list
・自身が所有する土地(plot)全ての権限を設定する(friend:フレンド、resident:街の住民、ally:同盟国の住人、outsider:その他全員)
(build:設置、destroy:破壊、switch:ボタン,スイッチの使用、itemuse:エンダーパールなどのアイテム使用)：
/res set perm 立場 権限 on/off
